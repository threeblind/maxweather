# 開発タスクと進捗管理
更新日: 2025-10-09

## ステータス定義
- **ToDo**: 要件は固まっているが着手していない。
- **In Progress**: 実装・調査中。担当と次アクションを明記。
- **Review (G)**: Gさんの検証・記事生成が必要な段階。
- **Blocked**: 外部要因で停止中。解除条件を記録。
- **Done**: 本番に反映済み。検証ログが残っている。
